# Profile

A flutter app that displays my Profile picture, name, and a link to my GitHub profile

## Getting Started

- Setup Flutter SDK 
- Setup Dart SDK
- Install Visual Studio Code (vs code)
- Install flutter and dart extensions in vs code
- Clone the repo
- Run the program

## Packages used 
- Svg
- webview_flutter
